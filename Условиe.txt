Условие задачи по созданию графовой базы данных "Музыкальная индустрия хип-хопа"

Формулировка задачи

Цель: Спроектировать и реализовать графовую базу данных для анализа взаимосвязей в музыкальной индустрии с использованием возможностей SQL Server для работы с графовыми данными.
Требования к реализации:

Создать схему графовой базы данных, включающую:
Таблицы узлов для основных сущностей: Артисты, Альбомы, Треки
Таблицы ребер для связей между ними: выпуск альбомов, принадлежность треков, участие в записи

Наполнить базу данных репрезентативными данными о современных хип-хоп исполнителях и их работах

Реализовать аналитические запросы, позволяющие:

Определять принадлежность треков к альбомам
Анализировать коллаборации между артистами
Исследовать связи в музыкальной индустрии через графовые паттерны
